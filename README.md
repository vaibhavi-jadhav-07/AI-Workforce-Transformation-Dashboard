# AI-Workforce-Transformation-Dashboard

Project Overview
This Power BI dashboard analyzes the impact of Artificial Intelligence on workforce transformation, automation risk, future career opportunities, and reskilling strategies.

The project provides insights into:

Workforce Risk Assessment
Future-Proof Careers
AI Exposure Analysis
Career Growth Potential
Workforce Transformation Recommendations

Dashboard Pages
1. Overview

Provides workforce summary metrics including salary, AI exposure, growth potential, and experience.

2. Risk Analysis

Identifies careers vulnerable to automation and highlights workforce risk distribution.

3. Future Careers

Explores future-ready careers, salary potential, and growth opportunities.

4. Workforce Transformation

Summarizes insights and provides recommendations for workforce adaptation.
